# 影像式穆勒矩陣計算系統

  這是一款由國立聯合大學光電工程所LAB405所開發的穆勒矩陣計算套件包，針對十六個元素進行計算及極化分解
  
  ## 穆勒矩陣運算
  
  主要來自橢圓偏振技術（**ellipsometry**）是一種多功能和強大的光學技術

  透過 **Python** 的程式計算可以將元件的極化特性轉變為更多分解後結果
